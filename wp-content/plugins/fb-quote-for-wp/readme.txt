FB Quote Plugin For WP lets people select text on your page and add it to their share on FB, so they can tell a more expressive story.


== Description ==

>FB Quote Plugin For WP lets people select text on your page and add it to their share on FB, so they can tell a more expressive story.

**Features**
- Quote to Facebook from Posts
- Quote to Facebook from Pages
- Quote to Facebook from Pages with shortcode `[fbq]`


**FB Quote Plugin Shortcode**

You can use `[fbq]` shortcode to add fb quote plugin anywhere you want.

Example Usage: `[fbq]`
This can be used inside the content area where ever the shortcodes work.

== Installation ==

Install/Activate the plugin and go to Settings > FB Quote Plugins page to configure it.

e.g.

1. Upload \`fb-quote-plugin-for-wp\` to the \`/wp-content/plugins/\` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the plugin and go to `Settings > FB Quote` page to configure it.
